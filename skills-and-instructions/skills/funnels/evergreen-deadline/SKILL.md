# Evergreen Launch Funnel

Build evergreen funnels that combine automated webinar/VSL content with deadline funnel logic to create urgency and drive conversions on-demand. Run continuously while creating real urgency through cart open/close windows.

## Core Objectives

- Create compelling automated content (webinar/VSL)
- Build desire and urgency through deadline logic
- Maximize conversions during cart open windows
- Run continuously with automated sequences
- Optimize for long-term performance

## Funnel Structure

### Entry Point (Registration)

**Registration Page:**
- **Headline:** Transformation promise or secret revealed
- **Subheadline:** What they'll learn or discover
- **Video Preview:** Teaser or thumbnail
- **Social Proof:** Attendee count, testimonials
- **CTA:** "Watch Free Training Now"

### Automated Content Delivery

**Webinar/VSL Flow:**
1. **Registration:** Capture lead and start deadline timer
2. **Content Delivery:** Pre-recorded, high-converting presentation
3. **Value Delivery:** Core teaching and transformation
4. **Offer Reveal:** Product or service introduction
5. **Cart Open:** Limited-time access window begins
6. **Cart Close:** Deadline triggers and urgency peaks

## Deadline Funnel Logic

### Cart Open Window

**Configuration:**
- Duration: Typically 24-72 hours from viewing offer
- Real countdown timer synced to lead's entry time
- Unique deadline per lead (no fake scarcity)
- Cookie and email tracking for accuracy

**Urgency Elements:**
- Personal deadline displayed on all pages
- Email reminders tied to individual countdown
- Bonus expiration at deadline
- Price increase after deadline

### Email Sequence (Automated)

**Post-Registration:**
1. Confirmation + access link
2. Reminder to watch if not completed
3. Value recap if watched

**Post-Offer View:**
1. Offer recap and bonuses reminder
2. Testimonials and proof
3. Objection handling
4. 24-hour warning
5. Final hours push
6. Cart closing (1 hour)
7. Cart closed notification

### Cart Close Handling

**When Deadline Expires:**
- Sales page shows "offer expired" message
- Option to get on waitlist
- Alternative offer or down-sell
- Re-engagement sequence for future opportunities

## Sales Page Elements

**Pre-Deadline:**
- Full offer with all bonuses
- Countdown timer prominently displayed
- Urgency messaging throughout
- Multiple CTAs

**Post-Deadline:**
- Clean "offer expired" page
- Waitlist signup option
- Alternative resources
- Future opportunity messaging

## Technical Requirements

**Deadline Tracking:**
- Individual countdown per lead
- Cookie-based tracking
- Email integration for reminders
- Fallback for cleared cookies

**Page Variations:**
- Pre-deadline version
- Post-deadline version
- Waitlist version

## Copywriting Style

- Conversational, urgent, conversion-focused
- Emphasize transformation and outcomes
- Use specific benefits and proof points
- Build urgency through deadline logic
- Create desire through value stacking

## Quality Checklist

- Is automated content compelling and conversion-focused?
- Does deadline logic create real urgency?
- Is cart open/close sequence optimized?
- Are reminders strategically timed?
- Will this convert consistently over time?
- Is the deadline system technically sound?
