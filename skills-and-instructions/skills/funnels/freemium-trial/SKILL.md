# Freemium / Free Trial Funnel

Build freemium/free trial funnels that convert prospects into free users, activate them quickly, and upgrade them to paid plans. Optimized for SaaS, software, or subscription-based businesses.

## Core Objectives

- Convert prospects into free users or trial signups
- Activate users quickly with time-to-value
- Demonstrate product value within trial period
- Upgrade free users to paid plans
- Retain and expand paid customers

## Funnel Structure

### Signup / Registration Page

**Above-the-Fold:**
- **Headline:** "Start Your Free [Trial/Account]"
- **Subheadline:** Value proposition and key benefit
- **Signup Form:** Minimal friction (email, password)
- **Social Proof:** User count, company logos, testimonials
- **CTA:** "Get Started Free" or "Start Free Trial"

**Value Proposition:**
- What they get for free
- Key features and benefits
- Time-to-value promise
- Upgrade benefits preview
- Trust indicators (security, privacy)

### Welcome & Onboarding

**Welcome Sequence:**
1. Welcome + getting started guide
2. Key feature walkthrough
3. First success milestone prompt
4. Advanced features introduction
5. Upgrade invitation (if appropriate)

**In-App Onboarding:**
1. Account setup and preferences
2. Core feature introduction
3. First action completion
4. Quick win achievement
5. Progress tracking

### Activation Strategy

**Key Activation Milestones:**
- Account setup complete
- First core action taken
- First value delivered
- First success achieved
- Engagement pattern established

**Time-to-Value Optimization:**
- Reduce steps to first value
- Progressive disclosure of features
- Contextual help and guidance
- Success celebration moments

### Upgrade Strategy

**Upgrade Triggers:**
- Feature limitation reached
- Usage threshold hit
- Time-based (trial ending)
- Value demonstration moment
- Success milestone achieved

**Upgrade Messaging:**
- **Value Stack:** What they get with paid
- **Feature Comparison:** Free vs. paid table
- **Social Proof:** Paid user testimonials
- **Urgency:** Trial ending, limited-time pricing
- **Risk Reversal:** Money-back guarantee

### Trial Ending Sequence

**Email Sequence:**
1. 7-day warning: Value recap + upgrade CTA
2. 3-day warning: Feature loss preview
3. 1-day warning: Final upgrade push
4. Expiration day: Last chance
5. Post-expiration: Win-back offer

## Copywriting Style

- Clear, benefit-focused, conversion-oriented
- Emphasize value and outcomes
- Use specific features and benefits
- Build trust through social proof
- Create urgency around upgrades

## Quality Checklist

- Does free offer provide genuine value?
- Is onboarding optimized for quick activation?
- Does upgrade path feel compelling and natural?
- Are upgrade triggers strategically placed?
- Will users feel product delivered value?
