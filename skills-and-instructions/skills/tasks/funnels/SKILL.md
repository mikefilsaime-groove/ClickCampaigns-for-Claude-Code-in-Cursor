# Funnel Pages & Landing Pages

Write high-converting landing pages for funnels, including lead capture pages, sales pages, OTO pages, and campaign pages. Apply proven principles from Russell Brunson, Mike Filsaime, Anik Singal, and Product Launch Formula.

## Core Objectives

- Maximize opt-ins, purchases, or CTA completions
- Capture and hold attention with curiosity and clarity
- Communicate emotional and strategic benefits
- Build urgency and trust quickly
- Drive action with scroll-based momentum

## Page Types

- **Lead Capture / Opt-In Page**
- **Sales Page (Long-Form or Short-Form)**
- **OTO / Upsell Page**
- **Webinar Registration Page**
- **Challenge Funnel Page**

## Core Page Structure

### 1. Headline + Subheadline (Above-the-Fold)

- Attention-grabbing, emotionally charged headline
- Clear subheadline expanding benefit/transformation
- Hook angle using numbers, outcomes, or paradigm shift
- [Hero Image or Video Thumbnail]

### 2. Lead Magnet or Offer Stack

- Bullet list (Feature ‚Üí Transformation framing)
- Immediate benefit + long-term gain
- [Supporting visuals, product mockups, icons]

### 3. Urgency & Scarcity

- Countdown timer
- Limited bonuses or enrollment caps
- Copy cues: "Only available this week"

### 4. Social Proof & Testimonials

- Quotes, screenshots, video testimonials
- Trusted client/media logos
- Early and mid-scroll placement

### 5. Offer Breakdown / Value Stack

- Visual component listing
- Real-world value and price anchor
- Bonus stack with stated value

### 6. Guarantee / Risk Reversal

- Money-back or conditional guarantee
- "Nothing to lose" reassurance
- Clear terms

### 7. Final CTA Section

- High-contrast CTA button
- Offer restated near button
- Trust badge or secure checkout icon

## Copywriting Style

- Clear, confident, results-focused tone
- Emotional momentum with conversational copy
- Short, rhythmic sentences
- Ellipses and curiosity loops
- Inspired by Brunson, Filsaime, Singal

## Quality Standards

**Headlines:** Stop the scroll
**Subheadlines:** Build desire
**Bullets:** Stack value
**CTAs:** Create urgency
**Overall:** Would you buy from it?

## Visual & UX Notes

- Mobile-first design
- Thumb-friendly buttons
- White space for flow
- Visual cues (arrows, animations)
- Sticky header or floating CTA

## Page-Specific Features

### OTO Pages
- Urgency-focused copy
- No exit links
- Price anchor
- Before/after contrast

### Opt-In Pages
- One CTA only
- Minimal friction (name + email)
- Zero distraction

### Webinar Registration
- Date/time lock-in
- Time zone display
- Instant confirmation
- Calendar add option

## Funnel Type Variations

| Type | Key Adjustments |
|------|-----------------|
| Discovery Call | Authority proof, low-friction booking |
| Product Launch | Pre-launch rhythm, cart open/close |
| Book Funnel | Chapter highlights, shipping value |
| Freemium | Time-to-value, plan comparison |
| Membership | Community, retention bonuses |
| Tripwire | Order bump + OTO profit stack |

## Guidelines

**Landing Page Principles (Per Russell Brunson, Mike Filsaime, Anik Singal):**

1. **One Page, One Goal** - Every page has ONE primary action
2. **Scroll-Based Momentum** - Copy builds desire as they scroll
3. **Feature ‚Üí Transformation** - Don't sell features, sell outcomes
4. **Price Anchor** - Always show value before price
5. **Risk Reversal** - Make buying feel safer than not buying

**Avoid Funnel Page Pitfalls:**
- Never: Multiple competing CTAs on opt-in pages
- Never: Exit links on OTO pages
- Avoid: Walls of text without visual breaks
- Skip: Generic headlines that could be anyone's

**Headline Formulas:**
- "How to [Outcome] Without [Pain Point]"
- "The [Number] Secrets to [Desired Result]"
- "Discover the [Method] That [Transformation]"
- "Finally: A [Solution] That Actually Works for [Audience]"

**Copywriting Style (Per Mike Filsaime):**
- Clear, confident, results-focused tone
- Emotional momentum with conversational copy
- Break up long sections with short, rhythmic sentences
- Use ellipses (...) and curiosity loops to keep users scrolling
- 3-5 lines maximum per paragraph

**Visual & UX Best Practices:**
- Mobile-first with thumb-friendly buttons
- White space and scroll indicators for flow
- Visual cues (arrows, animated elements) guide next steps
- Sticky header or floating CTA for long-form pages
- Video thumbnails with play button overlay

**Page-Specific Rules:**

**Opt-In Pages:**
- ONE CTA only (no navigation, no footer links)
- Minimal friction: first name + email maximum
- Zero distraction‚Äîremove everything that doesn't convert
- Promise immediate value (instant access)

**Sales Pages:**
- Multiple CTAs as they scroll (every 2-3 sections)
- Value stack before price reveal
- Testimonials distributed throughout
- Guarantee prominently displayed

**OTO Pages:**
- NO exit links (accept or decline only)
- Urgency-focused copy ("one-time only")
- Price anchor (show "regular price" first)
- Before/after emotional contrast
- Decline link should be understated

**Webinar Registration:**
- Date/time above the fold with timezone
- Calendar add buttons immediately after signup
- 3 Secrets / What You'll Learn format
- Countdown to live date

## Quality Checklist

- Does headline stop the scroll?
- Is value stack compelling?
- Is urgency real and believable?
- Are CTAs prominent and action-oriented?
- Would you convert from this page?
