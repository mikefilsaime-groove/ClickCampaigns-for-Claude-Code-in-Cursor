# SEO & Local Content

Create engaging, SEO-optimized content designed to rank well in search engines while delivering genuine value and engagement. Produce high-quality content that attracts organic traffic, encourages retention, and establishes authority.

## Core Objectives

- Rank well in search engines
- Deliver genuine value to readers
- Drive organic traffic
- Establish authority and expertise
- Convert readers to subscribers/customers

## SEO Content Structure

### Blog Title

- SEO-friendly and compelling
- Include primary keyword naturally
- Create curiosity or promise value
- Optimal length: 50-60 characters

### Opening Paragraph

- Hook that captures attention
- Include primary keyword naturally
- State what reader will learn
- Set expectations

### Body Content

**Heading Structure:**
- H2s for main sections (include keywords)
- H3s for subsections
- Logical hierarchy
- Scannable structure

**Content Guidelines:**
- Comprehensive topic coverage
- Specific examples and data
- Actionable steps
- Case studies where relevant

### Keyword Integration

**Primary Keywords:**
- In title, H1
- In first paragraph
- In 2-3 H2s
- Naturally throughout body
- In meta description

**Secondary Keywords:**
- In H3 subheadings
- Throughout body naturally
- In image alt text
- In internal/external links

**Avoid:**
- Keyword stuffing
- Unnatural placement
- Over-optimization
- Sacrificing readability

### Content Length

**Depth Mandate:**
- Cover topic comprehensively
- If 3,000 words needed, write 3,000
- If 800 words sufficient, don't pad
- Quality and depth over word count
- Every paragraph adds value

## On-Page SEO Elements

**Meta Title:**
- 50-60 characters
- Primary keyword included
- Compelling and clickable

**Meta Description:**
- 150-160 characters
- Primary keyword included
- Clear value proposition
- Call to action

**URL Structure:**
- Short and descriptive
- Include primary keyword
- Use hyphens between words
- Avoid parameters

**Image Optimization:**
- Descriptive file names
- Alt text with keywords
- Compressed for speed
- [Place Supported Image Here] markers

## Local SEO (If Applicable)

**Local Keywords:**
- City/region + service
- "Near me" variations
- Neighborhood terms
- Local landmarks

**Local Content:**
- Area-specific information
- Local case studies
- Community involvement
- Regional relevance

**Local Schema:**
- Business name, address, phone
- Service areas
- Business hours
- Reviews

## Copywriting Style

- Conversational, informative, authoritative
- Storytelling for engagement
- Emotional resonance
- Scannable formatting
- Reader-first approach

## Quality Standards

**Must Rank AND Engage:**
- No thin content
- No keyword stuffing
- Comprehensive coverage
- Unique insights
- Better than competing content

**Formatting:**
- Bold and italics for emphasis
- Bullet points and numbered lists
- Clear subheadings
- White space for readability
- Mobile-friendly structure

## Guidelines

**Avoid SEO Content Mistakes:**
- Never: Write for Google first, readers second
- Never: Keyword stuff unnaturally
- Avoid: Thin content that doesn't add value
- Skip: Copying competitors' structure exactly

**Keyword Integration Rules:**
- Primary keyword in first 100 words
- Use primary keyword 3-5 times per 1,000 words
- Variations and synonyms count as natural usage
- If it sounds weird, remove it

**Content Structure Best Practices:**
- H1: One per page, includes primary keyword
- H2: 5-10 per long-form article
- H3: Use for nested subtopics
- Lists and tables improve scannability

**Local SEO Essentials:**
- Include city/region in title, H1, and first paragraph
- Mention neighborhoods and landmarks
- Include NAP (Name, Address, Phone) consistently
- Add local schema markup

**Content Length by Intent:**
- Informational queries: 1,500-3,000+ words
- Transactional queries: 500-1,500 words
- Local service pages: 800-1,500 words
- Rule: Answer the query comprehensively

**Internal Linking:**
- Link to related content on your site
- Use descriptive anchor text
- 2-5 internal links per 1,000 words
- Avoid "click here" anchor text

**Featured Snippet Optimization:**
- Use question-based H2s
- Answer questions directly below heading
- Use numbered/bulleted lists
- Keep answers 40-60 words

**Engagement Signals:**
- Write compelling intros (hook in first 50 words)
- Use visuals every 300-500 words
- Include jump links for long content
- End with clear CTA

## Quality Checklist

- Does title include keyword and create curiosity?
- Is keyword integration natural?
- Is content comprehensive and valuable?
- Would reader share this content?
- Is this better than top 10 ranking articles?
- Is formatting scannable and engaging?
